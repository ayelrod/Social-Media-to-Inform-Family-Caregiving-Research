About

ALS Forums is an online forum where caregivers of patients with Amyotrophic Lateral Sclerosis (ALS) as well as patients post to give information, ask questions, and seek support. This page contains an NLP analysis of the Past Caregivers forum, used primarily by former caregivers of patients with ALS.

*Put link to download data set*

We performed natural language processing (NLP) on the data we gathered, including topic modeling and sentiment analysis. The following analysis is on posts containing the keywords "tele", "video", "virtual", "zoom", "skype", "phone". We separated the analysis into "Pre-Covid" (March 1, 2018 - February 29, 2020) and "Covid" (March 1, 2020 onwards). Further documentation can be found on the Documentation page.

Statistics:

The following statistics describe the ALS Past Caregivers forum between March 2018 and February 2022.

  • Unique users: 103
  • Number of Original Posts: 175
  • Number of Replies: 1826
  • Average Number of Posts per User (Including Replies): 850
  • Average Number of Original Posts per User: 1.7
  • Average Number of Replies per User: 17
  • Original Posts vs Replies Ratio: 0.095
  • Average Number of Words per Post: 90
  • Maximum Posts by One User in Timeframe: 546
  • Average Time Between Diagnosis and Joining the Forum: 377 days
  • Number of distinct cities: 96
